打造自己的独立博客

一：安装Hexo
高效的博客框架，用markdown（轻量级标记语言）解析文章，输出美观的静态页面，安装环境：node.js和git
1.检查mac是否安装好node.js和git
（1） 打开终端
node -v
v7.10.0
git —version
2.5.4

(2) 使用npm(js包管理工具)下载Hexo
npm install -g hexo-cli
下载出错：checkPermissions Missing write access to /usr/local/lib/node_modules[写入权限限制]
解决方案：
https://docs.npmjs.com/getting-started/fixing-npm-permissions
备注：通过命令把根目录都改为tony

(3)建站
找到一个文件夹，workspace-blog
执行命令，新建一个网站blog
hexo init blog
cd blog
npm install

建站完成后，会出现通用目录
_config.yml －配置参数
package.json －应用程序的信息，版本号等
scaffolds －模版文件
source －用户资源
_drafts
_posts
themes －主题文件，Hexo根据主题文件来生成静态页面

（4）写作
hexo new

(5) 服务器
hexo-server
